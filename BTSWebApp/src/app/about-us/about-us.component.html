Technologies Used



  1. MEAN Stack

      2. The MEAN stack is software stack for building web applications, back-ends and web sites. The "M" stands for MongoDB which is the database we use to store all our event and user information. The "E" stands for Express.js which is a modular web application framework package for Node.js. The next letter is "A" which stands for AngularJS. This is the front-end framework that we chose to use to flesh out our web application and create all it's functionality and usability. Lastly, the "N" - which stands for Node.js, is the application runtime that the MEAN stack operates on.

  2. Amazon Web Services (AWS)

      3. The AWS service that was used in this project was the S3 service. We used S3 buckets to store user profile pictures and user logo pictures.

  3. Google Maps API

      4. Google Maps API was integrated to show the location of an event on a map.
